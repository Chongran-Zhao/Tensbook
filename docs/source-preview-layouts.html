Option 1 — unified card · source auto-fits its content  ·  recommended

One bordered panel with a single divider. The source column is sized to the longest code line (capped), and the preview automatically takes whatever width is left — so short code gives the math more room, long code borrows it back. No fixed ratio to tune.

Option 3 — preview-first · source as a side rail

The result is the hero: a generous, centered preview with the source tucked into a slim tinted rail on the left (the same accent as a tens block in the editor). Best when the reader cares about the math and treats the code as reference.
