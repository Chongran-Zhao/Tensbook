A. Notebook Cards

soft structure, readable in preview
Problem
x^2\frac{df}{dx}+2f(x)=5x
ODE order 1 linear nonhomogeneous linear first-order

Normalize to standard form

\frac{df}{dx}+\frac{2}{x^2}f(x)=\frac{5}{x}

Use the integrating factor

\mu(x)=\exp\!\left(\int\frac{2}{x^2}\,dx\right)=e^{-2/x}

Integrate the product derivative

\frac{d}{dx}\big(\mu f\big)=\mu\,\frac{5}{x}
\mu f=\int \mu\,\frac{5}{x}\,dx+C_1
Final answer
f(x)=\frac{5\int e^{-2/x}\frac{1}{x}\,dx+C_1}{e^{-2/x}}

E. Lecture Blocks

good when notes contain prose between equations

Problem

x^2\frac{df}{dx}+2f(x)=5x

This is a first-order linear ODE.

After division by x^2, the coefficient is p(x)=2/x^2.

Integrating factor

\mu(x)=\exp\!\left(\int p(x)\,dx\right)=e^{-2/x}

One integration gives the solution family

\mu f=\int \mu\,\frac{5}{x}\,dx+C_1
f(x)=e^{2/x}\left(5\int\frac{e^{-2/x}}{x}\,dx+C_1\right)
