# Tensbook v1.1.2

Tensbook v1.1.2 is a focused reliability release for the live notebook preview and Markdown renderer.

## Fixes

- Prevent older asynchronous evaluations from overwriting the preview for a newer edit or document.
- Surface current Tauri invocation failures in the preview instead of leaving an unhandled promise rejection.
- Preserve Markdown source order when ordinary links and embedded `data:image` images appear on the same line.

## Quality

- Added regression coverage for out-of-order evaluation results, invalidated in-flight work, backend rejection handling, and mixed link/image Markdown.
- Passed 188 Rust workspace tests and 22 headless UI tests.
- Completed desktop QA for rapid live-preview updates, Markdown rendering, file open/save, and Markdown/PDF export.

## Upgrade

```sh
brew upgrade --cask tensbook
```

There are no `.tens` language or notebook format changes in this release.
